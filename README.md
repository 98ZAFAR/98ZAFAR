1. **[Pen-Mail](https://github.com/98ZAFAR/pen-mail)**
   A Slowly-inspired pen-pal web app built with the MERN stack and TypeScript. Features intentional delivery delays based on distance, user matchmaking, and clean UI/UX for meaningful communication.

2. **[Chat-In](https://github.com/98ZAFAR/Chat-In)**
   A real-time chat application using the MERN stack and Socket.io. Supports instant 1-to-1 messaging, secure authentication, and scalable full-stack architecture.

3. **[Disposable Email Service](https://github.com/98ZAFAR/dis-mail-backend)**
   Core API for a disposable email system integrated with a custom-built **[SMTP mail server](https://github.com/98ZAFAR/mail-server)**. Manages temporary mailboxes, secure sessions, and attachments using Node.js, PostgreSQL, and Redis.

4. **[Ngen2D](https://github.com/98ZAFAR/Ngen2D)**
   A from-scratch 2D physics engine written in C++. Features rigid body dynamics, collision detection/resolution, and rendering via SDL2.

5. **[BuzzScript](https://github.com/98ZAFAR/BuzzScript)**
   A custom toy programming language and interpreter built with TypeScript to explore language design and evaluation mechanics.

6. **[AptiCrack](https://github.com/98ZAFAR/AptiCrackFrontend)**
   A comprehensive exam-taking platform frontend built with React.js.
